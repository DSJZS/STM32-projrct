/*********************************************************************************************/
[*]程序简介

-编码方式( Encoding ): UTF-8
-工程名称:二轮平衡小车
-MDK版本:5.24.2.0
-ST固件库版本:3.5.0
-单片机型号:STM32F103C8T6

/*********************************************************************************************/
[*]功能简介:

-1.运用PID使得二轮小车保持平衡
-2.通过按键双击进入调试模式显示当前数据(调试模式不能修改数据，只能看LOL)
-3.可以通过蓝牙连接，远程操控小车移动

/*********************************************************************************************/

-外设-                -外设的引脚-             -电压、GPIO引脚等-
