[*]程序简介

-编码方式( Encoding ): UTF-8
-工程名称:二轮平衡小车的蓝牙遥控器
-MDK版本:5.24.2.0
-ST固件库版本:3.5.0
-单片机型号:STM32F103C6T6 (注意这个项目的设备不是C8T6,是C6T6,按需求修改)

-1.远程遥控二轮平衡小车移动
